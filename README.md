# StockManager

A szoftver célja egy olyan web alapú alkalmazás fejlesztése, amely lehetőséget biztosít a felhasználók számára amerikai részvények technikai és fundamentális elemzésére, valamint saját részvényportfólió összeállítására és nyomon követésére.

---

## Technológiai környezet

- Frontend: **React**
- Backend: **C# .NET 9 - Entity Framework Core Web API**
- Adatbázis: **SQLite**
- Fejlesztői eszközök: Visual Studio Code, Visual Studio 2022
- Egyéb: **Fetch API**

--

## Főbb funkciók

- Részvények böngészése, adatok megtekintése elemzéshez
- Részvény portfólió összeállítása, változások nyomon követése
- Részvényekre vonatkozó tranzakciók rögzítése
- Portfólió teljesítményének kiértékelése
- Részvény-figyelő lista létrehozása
- Oktatói anyagok megtekintése
- Jelentős árfolyamváltozáson átmenő részvények nyomon követése
- Árfolyam értesítők célárak elérésekor

--

## Felhasználói szerepkörök

1. Adminisztrátor szerepkör
Az adminisztrátor szerepkör célja, hogy lehetőséget biztosítson a felhasználók- és részvények kezelésére, valamint a szoftverben megtalálható oktatói modul szerkesztésére. 

2. Befektető szerepkör
A befektető szerepkör célja, hogy lehetőséget biztosítson a felhasználóknak, saját portfóliójuk kezelésére és részvények elemzésére.
